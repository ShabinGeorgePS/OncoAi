# Dataset Information

## Source
Reference repo: https://github.com/PrateekDutta2001/Oral-Cancer-Detection
Dataset link: https://www.kaggle.com/datasets/zaidpy/oral-cancer-dataset

## Classes
| Class      | Folder Name | Total Images |
|------------|-------------|--------------|
| Non-Cancer | non-cancer/ | 500          |
| Cancer     | cancer/     | 250          |

## Preprocessing
- Image size: 224 x 224 pixels
- Normalization: divided by 255
- Split: 70% train / 15% val / 15% test
