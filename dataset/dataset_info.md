Downloaded from: https://www.kaggle.com/datasets/zaidpy/oral-cancer-dataset

|      Class       | Folder Name | Image Count |
|------------------|-------------|-------------|
| Oral Cancer      |   `CANCER`  |    500      |

| **Total** | | **750** |

---

## Class Imbalance

The dataset is **imbalanced** — Cancer class has 2× more images than Non-Cancer.

|        Metric        |                             Value                               |
|----------------------|-----------------------------------------------------------------|
| Imbalance Ratio      |                2:1 (Cancer : Non-Cancer)                        |
| Handling Strategy    |          Data Augmentation on training set                      |
| Augmentation Applied |    Rotation, Zoom, Horizontal Flip, Shear, Width/Height Shift   |

---

## Image Details

| Property        |                Value                        |
|-----------------|---------------------------------------------|
| Original Format |           JPG / JPEG / PNG                  |
| Original Sizes  |     Variable (multiple resolutions)         |
| Resized To      | 224 × 224 pixels (for EfficientNetB0 input) |
| Color Mode      |               RGB (3 channels)              |
| Normalization   |  Pixel values ÷ 255 → range [0.0, 1.0]      |

> Keras assigns class indices alphabetically. C comes before N, so CANCER = 0, NON CANCER = 1.
